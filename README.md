# SMSChatbot using Open AI and Twilio 

This app was deployed on Amazon Elastic BeanStake, which is a PaaS provided by AWS for developers to host apps developed using various technologies like Python 
Node-js, Docker, ... etc

# Technologies:
1. Flask
2. Open-ai API
3. Twilio
4. ngrok
5. AWS DynamoDB
6. Elastic Beanstake
